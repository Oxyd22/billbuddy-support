## Datenschutz / Privacy Policy

BillBuddy speichert alle Daten ausschließlich lokal auf deinem Gerät.
Es werden keine persönlichen Daten gesammelt, gespeichert oder übertragen.
Es gibt keine Benutzerkonten, keine Cloud-Verbindung und keine Analyse-Tools.

This app does not collect any personal data.
All information entered (names, orders, amounts) remains exclusively on
your device and is never shared with third parties. There are no user accounts,
no cloud connection, and no analytics.
